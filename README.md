# DrumCloud

DrumCloud by Fuimadane is a granular sample instrument, built with DPF and released in CLAP, VST2, VST3 and LV2 formats.

Load your own samples directly from the plugin UI and explore them with multiple scan modes, waveform preview, Sync X control, and a dark custom interface with Fuimadane artwork in the background.

Features:

- CLAP, VST2, VST3, and LV2

- Load your own WAV, FLAC and MP3 samples

- Granular sample playback

- Multiple scan modes

- Waveform preview

- Sync X control

- Saved state: host remembers sample, mode and settings

Original repository: https://github.com/Chmod666music/DPF
